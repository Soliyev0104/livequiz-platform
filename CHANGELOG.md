# Changelog

## v1.0.0 — Course submission

- Real-time multiplayer quiz rooms.
- FastAPI REST API and WebSocket gameplay.
- Postgres relational schema with Alembic migrations.
- Redis live room state, cache, leaderboard, and pub/sub.
- Redpanda event stream and ClickHouse analytics.
- Nginx gateway with two API replicas.
- Snowflake-style ID generator from scratch.
- Grafana observability stack.
